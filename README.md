A full-stack task management system with secure user authentication

Description:
Developed a full-stack e-Todo Application that allows users to securely manage their daily tasks. The application enables authenticated users to create, update, view, and delete todos using a modern microservice-style backend and a responsive Angular frontend.

Key Responsibilities & Features

* Built RESTful APIs using Spring Boot, enabling seamless CRUD operations for todo management.

* Implemented Spring Security with JWT authentication, ensuring only authorized users can access and modify their tasks.

* Utilized Spring Data JPA with H2 in-memory database for fast data processing and simplified testing.

* Designed a clean and interactive UI using Angular, supporting real-time form validation and smooth user experience.

* Implemented layered architecture (Controller → Service → Repository) ensuring clean code separation and maintainability.

* Enabled CORS configuration to ensure secure communication between Angular frontend and Spring Boot backend.

Tech Stack :
* Backend: Java, Spring Boot, Spring Security (JWT), Spring Data JPA, H2 Database
* Frontend: Angular
* Tools: Maven, Postman, Git
